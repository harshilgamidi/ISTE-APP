## Run Locally

Clone the project

```bash
  git clone https://github.com/harshilgamidi/ISTE-APP
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the frontend

```expo
  npx expo start
```

Start the Backend

```npm
  cd Backend
  npm start start
```
